# Pinakes - Documentazione

Benvenuto nella documentazione tecnica di **Pinakes**, il sistema di gestione bibliotecaria open source.

## A chi è rivolta

Questa documentazione è destinata a:
- **Operatori bibliotecari** che utilizzano quotidianamente l'applicazione
- **Amministratori di sistema** responsabili della manutenzione
- **Sviluppatori** che vogliono estendere le funzionalità

## Sezioni Principali

### Guida Operatore
Istruzioni pratiche per le operazioni quotidiane: gestione catalogo, prestiti, prenotazioni, utenti.

### Amministrazione
Configurazione del sistema, aggiornamenti, gestione plugin e backup.

### Riferimento Tecnico
Documentazione per sviluppatori: installazione, API, sviluppo plugin.

## Requisiti Minimi

| Componente | Versione |
|------------|----------|
| PHP | 8.0+ |
| MySQL/MariaDB | 5.7+ / 10.3+ |
| Web Server | Apache/Nginx |
| Estensioni PHP | mysqli, json, mbstring, curl, zip |
