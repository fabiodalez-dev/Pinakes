# Pinakes - Documentation

Welcome to the technical documentation for **Pinakes**, the open source library management system.

## Who Is This For

This documentation is intended for:
- **Library operators** who use the application daily
- **System administrators** responsible for maintenance
- **Developers** who want to extend functionality

## Main Sections

### Operator Guide
Practical instructions for daily operations: catalog management, loans, reservations, users.

### Administration
System configuration, updates, plugin management and backup.

### Technical Reference
Documentation for developers: installation, API, plugin development.

## Minimum Requirements

| Component | Version |
|-----------|---------|
| PHP | 8.0+ |
| MySQL/MariaDB | 5.7+ / 10.3+ |
| Web Server | Apache/Nginx |
| PHP Extensions | mysqli, json, mbstring, curl, zip |
