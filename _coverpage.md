# Pinakes

> Sistema di Gestione Bibliotecaria Open Source

**Versione 0.7.x**

- Catalogo bibliografico e sistema prestiti/prenotazioni configurabile
- Modalità privata (accesso riservato agli utenti registrati)
- **Mobile API + app Android** — catalogo, prestiti e prenotazioni dal telefono
- Archivi ISAD(G) / ISAAR(CPF) · RiC-O JSON-LD · MARCXML / EAD3 / METS
- Classificazione Dewey integrata
- Sistema plugin estensibile (18 plugin bundled)
- Interoperabilità: SRU/Z39.50, OAI-PMH, NCIP, BIBFRAME, VIAF, IIIF, ResourceSync
- Scraping metadati (Open Library, Google Books, Discogs, MusicBrainz, Deezer, BNF/UNIMARC)
- Installazione one-click, immagine **Docker** ufficiale, aggiornamenti dal pannello admin
