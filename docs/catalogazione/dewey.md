# 📊 Classificazione Decimale Dewey (DDC)

## Cos'è la Classificazione Dewey?

La **Classificazione Decimale Dewey** (DDC, Dewey Decimal Classification) è il sistema di organizzazione biblioteconomica più diffuso al mondo, creato da Melvil Dewey nel 1876. Organizza la conoscenza in 10 classi principali numerate da 000 a 900, ogni classe si divide in divisioni e sezioni in modo gerarchico.

**Obiettivo:** Permettere di trovare libri per argomento in modo universale e standardizzato.

## 🎯 Le 10 Classi Principali

### 000-099: Informatica, Informazione e Opere Generali

**Ambito:** Computer science, biblioteche, enciclopedie, giornalismo, editoria

**Divisioni principali:**
- **000-009** Computer science, informazione e opere generali
- **010-019** Bibliografia
- **020-029** Biblioteconomia e scienze dell'informazione
- **030-039** Enciclopedie generali
- **040-049** (Non assegnato)
- **050-059** Riviste, pubblicazioni seriali
- **060-069** Associazioni, organizzazioni e musei
- **070-079** Giornalismo, editoria, mass media
- **080-089** Opere collettanee generali
- **090-099** Manoscritti e libri rari

**Esempi pratici:**
- `000` - Generalità informatiche
- `025.04` - Sistemi di archiviazione e recupero informazioni
- `070.5` - Editoria

## 🔧 Come Funziona in Pinakes

### Sistema JSON (dalla v0.3.0)

Pinakes memorizza le classificazioni Dewey in **file JSON modificabili**:

```
locale/
├── dewey_it_IT.json  (1.369 voci in italiano)
└── dewey_en_US.json  (1.369 voci in inglese)
```

**Struttura file JSON:**
```json
{
  "classes": [
    {
      "code": "800",
      "name": "Letteratura",
      "divisions": [
        {
          "code": "850",
          "name": "Letteratura italiana",
          "sections": [
            {
              "code": "851",
              "name": "Poesia italiana"
            },
            {
              "code": "853",
              "name": "Narrativa italiana"
            }
          ]
        }
      ]
    }
  ]
}
```

### Interfaccia Utente

**Durante l'inserimento libro:**
1. Vai a Dashboard → Libri → + Nuovo Libro
2. Scorri fino a "Classificazione Dewey"
3. **Selezione a cascata:**
   - Primo dropdown: Scegli la **classe** (es: 800 - Letteratura)
   - Secondo dropdown: Scegli la **divisione** (es: 850 - Letteratura italiana)
   - Terzo dropdown (opzionale): Scegli la **sezione** (es: 853 - Narrativa)
4. Il codice finale viene salvato automaticamente

**Non devi ricordare i numeri!** L'interfaccia mostra i nomi completi e tu selezioni da menu.

### Plugin Dewey Editor

Dashboard → Plugin → Dewey Editor

**Funzionalità:**
- 📖 **Visualizza albero completo** delle classificazioni
- ➕ **Aggiungi** nuove classificazioni personalizzate
- ✏️ **Modifica** nomi esistenti
- 🗑️ **Elimina** categorie non utilizzate
- 📥 **Importa** set di classificazioni da JSON
- 📤 **Esporta** configurazione personalizzata

**Quando usarlo:**
- Vuoi aggiungere sottocategorie specifiche
- Hai bisogno di classificazioni molto specifiche
- Gestisci una biblioteca specializzata (es: solo scienze)

## 💡 Come Classificare un Libro

### Metodo 1: Usa lo Scraping (Più Semplice)

Quando inserisci un libro con ISBN, il sistema:
1. Scarica i metadati automaticamente
2. Recupera la classificazione Dewey se disponibile nella fonte
3. La compila automaticamente

**Fonti con Dewey:**
- ✅ SBN (Sistema Bibliotecario Nazionale) - Molto affidabile
- ⚠️ Google Books - A volte disponibile
- ❌ Open Library - Raramente disponibile

### Metodo 2: Classifica Manualmente

**Step by step:**

1. **Identifica l'argomento principale del libro**
   - Di cosa parla? (es: romanzo italiano)

2. **Scegli la classe principale (centinaia)**
   - Esempio: Letteratura = 800

3. **Restringi alla divisione (decine)**
   - Esempio: Letteratura italiana = 850

4. **Opzionale: Scegli la sezione (unità)**
   - Esempio: Narrativa italiana = 853

**Risultato finale:** `853`

## 🎯 Best Practices

### ✅ Da Fare

1. **Sii coerente**: Usa sempre lo stesso livello di specificità per argomenti simili
2. **Usa il livello giusto**: Non serve sempre arrivare alla sezione (3 cifre), a volte la divisione (2 cifre) basta
3. **Consulta online**: Se in dubbio, cerca "classificazione dewey [argomento libro]"
4. **Controlla libri simili**: Vedi come altri hanno classificato libri dello stesso genere
5. **Documenta scelte**: Usa il campo "Note" per spiegare scelte non ovvie

### ❌ Da Evitare

1. **Non inventare codici**: Usa solo codici standard esistenti
2. **Non mescolare sistemi**: Non usare Dewey + altri sistemi insieme
3. **Non essere troppo specifico**: Se non sei sicuro, resta più generico
4. **Non ignorare l'argomento principale**: Se un libro parla di più cose, classifica per l'argomento PRINCIPALE

## ❓ Domande Frequenti

**D: La classificazione Dewey è obbligatoria?**
R: No, è opzionale. Molte piccole biblioteche usano solo generi letterari e collocazione fisica.

**D: Posso modificare le categorie Dewey?**
R: Sì, tramite il plugin Dewey Editor o modificando direttamente i file JSON.

**D: Cosa faccio se un libro copre più argomenti?**
R: Classifica per l'argomento PRINCIPALE. Se proprio necessario, aggiungi una nota nel campo "Note".

**D: Devo usare sempre 3 cifre?**
R: No. Puoi fermarti alla classe (1 cifra, es: 800) o divisione (2 cifre, es: 850). Più cifre = più specifico.

**D: Posso cercare libri per classificazione Dewey?**
R: Non direttamente nel catalogo pubblico, ma puoi filtrare nel pannello admin o esportare in CSV e filtrare.

**D: Il sistema Dewey è multilingua?**
R: Sì! I file JSON sono tradotti in italiano e inglese. Cambiano i nomi ma i codici restano uguali.
