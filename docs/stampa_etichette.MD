# Stampa Etichette Libri - Guida Completa per Utenti

## Introduzione: Perché Serve Stampare Etichette?

Le etichette sono **etichette adesive** che attacchi sul dorso dei libri per:
- **Trovare rapidamente** i libri sugli scaffali
- **Organizzare** la biblioteca in modo professionale
- **Identificare** ogni libro univocamente
- **Risparmiare tempo** durante le ricerche

---

## Formati Disponibili (Scegli il Tuo!)

| **Formato** | **Dimensioni** | **Dove si usa** | **Consigliato per** |
|-------------|----------------|------------------|---------------------|
| **Standard** | 25×38mm | Dorso libro | **Tutti i libri** |
| **Medio** | 50×25mm | Dorso grande |  Libri spessi |

>  **Consiglio**: Usa il formato **25×38mm** per tutto. È lo standard universale!

## Come Stampare un'Etichetta (Passo-Passo)

### Passo 1: Trovare il Libro
1. **Vai su**: Dashboard → **Libri**
2. **Cerca** il libro che ti serve
3. **Clicca** sul **titolo** del libro
4. **Vedrai**: La scheda completa del libro

### Passo 2: Generare l'Etichetta
1. **Clicca** il bottone **"Stampa Etichetta"**
2. **Attendi**: 2-3 secondi per generare il PDF
3. **Si apre**: PDF con l'etichetta pronta

### Passo 3: Stampare Correttamente
**Impostazioni Obbligatorie**:
- **Dimensione carta**: Personalizzata 25×38mm
- **Ridimensionamento**: 100% (nessuno)
- **Margini**: Nessuno
- **Orientamento**: Verticale

---

## Cosa Contiene l'Etichetta

### Formato Standard (25×38mm)
```
┌─────────────────────┐
│  Biblioteca         │  ← Nome tua biblioteca
│  Marconi            │
├─────────────────────┤
│                     │
│  Il Nome della      │  ← Titolo (abbreviato se lungo)
│  Rosa               │
│                     │
│  Umberto Eco        │  ← Autore principale
│                     │
│  ▐▌▐▌▐▌▐▌▐▌        │  ← Codice a barre (ISBN)
│                     │
│      A.2.15        │  ← Posizione fisica
└─────────────────────┘
```

### Cosa Significano i Numeri
- **A.2.15** = Scaffale A, Mensola 2, Posizione 15
- **Codice a barre**: Per scanner futuri (se avrai lettori)

## Istruzioni Dettagliate per Stampare

### Stampante Laser (Consigliata)
1. **Apri** il PDF generato
2. **Vai su**: File → Stampa
3. **Imposta**:
   - Carta: Etichette 25×38mm
   - Qualità: Alta
4. **Stampa** una prova su carta normale
5. **Se è perfetta**: stampa sull'adesivo

### Stampante Inkjet
1. **Usa** carta adesiva **resistente all'acqua**
2. **Lascia asciugare** 5 minuti prima di applicare
3. **Evita** di toccare la stampa appena fatta

---

## Come Applicare le Etichette

### Preparazione
1. **Pulisci il dorso** del libro con panno asciutto
2. **Togli polvere** per far aderire meglio
3. **Rimuovi** etichette vecchie se presenti

### Posizione Corretta
- **Dove**: Dorso del libro, 2-3 cm dal fondo
- **Allineamento**: Centrato e dritto
- **Pressione**: Premi bene per 5 secondi

### Per Libri Spessi
- **Posizione**: Più in alto per visibilità
- **Orientamento**: Orizzontale se necessario

## Configurare il Formato Etichette

### Guida Dettagliata alle Impostazioni

#### Passo 1: Accedi alle Impostazioni
```
Dashboard →  Impostazioni (in fondo al menu)
```

#### Passo 2: Trova la Sezione Etichette
```
Cerca nella pagina Impostazioni:
-  "Etichette" oppure
-  "Label Settings" oppure
-  "Stampa"
```

#### Passo 3: Scegli il Formato
```
Opzione 1: 25×38mm (CONSIGLIATO)
-  Standard universale
-  Perfetto per la maggior parte libri
-  Carta adesiva ampiamente disponibile
-  Usa questo per tutto

Opzione 2: 50×25mm (Alternativo)
- Per libri molto spessi
- Etichetta più grande
- Visibilità migliore

Opzione 3: 70×36mm (Speciale)
- Per etichette molto grandi
- Per libri antichi/rari
- Uso limitato
```

#### Passo 4: Configura la Carta
```
Tipo di Carta:
□ Adesiva Standard (consigliata)
  - Carta adesiva normale
  - Usa per stampanti laser o inkjet

□ Adesiva Resistente all'Acqua
  - Per ambienti umidi
  - Per biblioteche con problemi umidità
  - Leggermente più costosa

□ Carta Normale (solo prova)
  - Per test prima di stampare su adesiva
  - Non aderisce
  - Usa per verificare allineamento
```

#### Passo 5: Impostazioni Avanzate (Facoltativo)
```
Font: Lasciare di default (Arial o sans-serif)
Dimensione Font: Automatica (sconsigliato cambiare)
Margini: Automatico (standard)
Orientamento: Verticale per 25×38mm
            Orizzontale per 50×25mm
```

#### Passo 6: Salva Configurazione
```
Clicca:  "Salva Impostazioni"
        oppure "Save Settings"
Vedrai:  "Configurazione salvata con successo!"
        oppure "Settings updated!"
```

#### Passo 7: Verifica
```
1. Vai a un libro
2. Clicca  "Stampa Etichetta"
3. Controlla il PDF generato
4. Verifica che il formato sia corretto
5. Se no, ripeti i passi 1-6
```

### Troubleshooting Impostazioni

#### "Non trovo il menu Impostazioni"
- Verifica di essere loggato come admin
- Scorri in fondo al menu a sinistra
- Cerca il simbolo 

#### "L'opzione Etichette non è visibile"
- Potrebbe essere sotto "Stampa" o "Configurazione"
- Cerca parole chiave: "label", "etichetta", "print"

#### "Le impostazioni non si salvano"
- Controlla la connessione internet
- Riprova il salvataggio
- Contatta l'amministratore se persiste

#### "Le etichette stampate non corrispondono al formato selezionato"
- Verifica le impostazioni della stampante (100%, no ridimensionamento)
- Controlla il tipo di carta caricata
- Ripeti i passi di configurazione

## Problemi Comuni e Soluzioni

### "L'etichetta stampa troppo piccola/grande"
- **Verifica**: Impostazioni di stampa al 100%
- **Controlla**: Dimensione carta selezionata correttamente

### "Il codice a barre non appare"
- **Motivo**: Il libro non ha ISBN
- **Soluzione**: L'etichetta avrà solo testo, va bene lo stesso

### "L'etichetta non si attacca"
- **Motivo**: Dorso del libro sporco o unto
- **Soluzione**: Pulire con alcool prima di applicare

### "L'etichetta è storta"
- **Consiglio**: Usa una riga come guida per allineare
- **Alternativa**: Rimuovi delicatamente e riapplica

## Consigli per l'Efficienza

### Stampare in Batch
1. **Raccogli** 10-20 libri nuovi
2. **Stampa** tutte le etichette insieme
3. **Applica** subito le etichette stampate
4. **Segna** su un foglio i libri completati

### Organizzazione
- **Per scaffale**: Stampare prima tutte le A, poi B, poi C...
- **Per data**: Stampare etichette per libri arrivati oggi
- **Per genere**: Etichette di narrativa, poi saggistica...
