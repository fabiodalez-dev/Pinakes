## 🎯 Quick Start

**Cambiare tema rapidamente:**

1. **Dashboard → Impostazioni → Tab "Temi"**
2. Scegli uno dei 10 temi dalla galleria
3. Clicca "Applica"
4. ✅ Cambiamento immediato!

**Tempo:** 10 secondi

**Personalizzare homepage:**

1. **Dashboard → Impostazioni → Tab "CMS"**
2. Modifica titolo hero, features, CTA
3. Carica immagine background
4. Salva
5. ✅ Vedi subito sulla homepage pubblica!

**Tempo:** 5 minuti

## 🔧 Personalizzazione Avanzata

### Custom CSS

Per utenti avanzati, puoi aggiungere CSS personalizzato:
- Dashboard → Impostazioni → Avanzate → Custom CSS
- Inserisci il tuo CSS
- Salva
- Viene applicato globalmente

### Sezioni Riorganizzabili

Dalla v0.4.0, puoi riorganizzare le sezioni della homepage con drag & drop:
1. Dashboard → CMS → Homepage
2. Trascina le sezioni nell'ordine desiderato
3. Toggle per abilitare/disabilitare
4. Salva layout

**Sezioni disponibili:**
- Hero Section (ricerca e statistiche)
- Features (vantaggi biblioteca)
- Ultimi Libri Aggiunti
- Categorie
- Eventi Prossimi
- Call to Action

## ❓ Domande Frequenti

**D: Posso creare un tema completamente personalizzato?**
R: Sì, tramite il Custom Theme Editor o aggiungendo CSS custom.

**D: Le modifiche ai temi sono immediate?**
R: Sì, cambiano immediatamente per tutti gli utenti.

**D: Posso tornare al tema default?**
R: Sì, basta selezionare "Sky Blue" (tema predefinito).

**D: Il logo viene mostrato anche nelle email?**
R: Sì, il logo appare nell'header delle email di sistema.
